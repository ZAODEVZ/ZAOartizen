# ZAO on Artizen - operating rhythm + metrics

How to run the Artizen effort as a steady cadence, not a one-off push. Pair with the kit (outreach,
submissions, daily spotlights) and the research (fund directory, strategy docs).

## Daily

- Post one project spotlight (kit/daily-spotlights.md). Re-check the live fund first for which projects
  have match remaining; tag the creator; reshare their post same day.
- Collect/boost at least one project in the ZAO fund or an adjacent fund (lead from the front).

## Weekly

- Check the active fund drive + deadline (drives run weekly with escalating match). Time a member rally
  (kit/outreach-drafts.md #4) to the drive close so the ZAO fund's match gets unlocked.
- Cross-back 2-3 projects in an adjacent fund (Greenpill, Funding the Commons), then DM that director
  about cross-curation (kit/outreach-drafts.md #3).
- Post a roundup: "this week in the ZAO fund" - who climbed, what unlocked, what's next.

## Per season (Curate -> Compete)

- Curation: rally the 188 members to vote ZAO projects into the top 30%. Onboard the cohort
  (kit/submission-template.md): WaveWarZ, SongJam, ZAOstock, Zaoville, Thy Revolution.
- Competition: drive artifact sales; stack each project into every fund it fits (research/fund-directory.md).
- Close: thank backers, document outcomes, line up next season's cohort.

## One-time setup (gated on the René call)

- Get on Console (the Artizen social layer).
- Apply + create the ZAO Festivals Fund (kit/fund-proposal.md, kit/rene-call-brief.md).
- Pursue the $50K community-fund seed + a ZAO-treasury Founding Sponsor stake.

## Metrics to track (snapshot weekly)

| Metric | Where | Why |
|---|---|---|
| ZAO Fund standing (rank/score/raised) | artizen.thezao.com | Are we climbing? Moves every drive. |
| Match remaining in the ZAO Fund | fund page | Unspent match = money left on the table; rally to unlock it. |
| ZAO projects' sales + match | each project | Which creators are converting; who needs a push. |
| # ZAO projects curated | fund page | Portfolio coverage - aim to grow each season. |
| Artizen Endowment $ | artizen.fund | The platform's health + whether René's flywheel is tracking ($4M -> $10M EOY goal; ~$14M as of Jun 2026). |
| Active drive + deadline | fund page | Time rallies + posts to it. |
| ZAO Festivals Fund status | (post-launch) | Once live: its own standing + project count. |

## Cadence rule

The platform rewards showing up. A community that posts daily, rallies weekly, and curates every season
beats one that does a big push and goes quiet. Consistency is the edge - ZAO's 188 members are the engine.
