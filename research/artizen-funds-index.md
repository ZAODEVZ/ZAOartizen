# Artizen funds index - where to submit ZAO projects for stacked match

**Why this exists:** the single biggest match lever on Artizen is qualifying for MULTIPLE funds. Every $10
Artifact sale unlocks $10 of match from EACH fund backing the project - so a project curated by 3 funds
matches 3x on the same sales. This index maps every Artizen fund + which ZAO-type projects fit it, so we submit
each project to every fund it qualifies for.

**Status: v1 (2026-06-27), being expanded.** The funds directory (artizen.fund/index/matchfunds) is Bubble.io
client-rendered and not fetchable, so this is built from web search + the Artizen newsletter + a background
research pass. Confidence-tagged. A screenshot of the live /matchfunds page from Zaal would complete it fastest.

## The stacking rule (verified)

- Submit each project to the ZAO Fund AND every other fund it qualifies for.
- Each fund = its own 1:1 match pool on top of the same sales.
- Plus the Artizen Endowment adds $1+ on top -> ~$3+ per $1 baseline, more with multiple funds.
- (Match per fund is capped; unused match rolls to next season.)
