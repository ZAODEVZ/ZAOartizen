# Artizen Fund Directory + ZAO Stacking Map

The community-fund landscape on Artizen, who runs them, and exactly which ZAO projects should curate
into which funds to stack match. Snapshot 2026-06-13 (rendered live from artizen.fund/index/matchfunds
+ the Greenpill fund page). Numbers move daily mid-drive - re-render before quoting.

## Why this matters

A project curated into N funds unlocks $1 of match from EACH per $1 of artifact sales. The whole game
is getting each ZAO project into every fund it credibly fits. This doc is the target list.
