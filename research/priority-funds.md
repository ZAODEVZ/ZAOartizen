## The Festivals Fund re-decision (recommendation)

Given the corrected landscape (research-audit.md): **don't create a new ZAO Festivals Fund yet. Curate
ZAOstock + Zaoville INTO the funds that already exist and already host ZAO's music projects.**

Why:
- We're Loud (DIY festivals), Global Music, and Greenpill (regen gatherings) already cover music events -
  and Global Music already hosts 6 ZAO projects. The "white space" that justified a new fund is gone.
- A new fund is an 80th fund in a crowded field + real ongoing curation work. Stacking into 3-4 existing
  funds gets ZAOstock the same match with none of the overhead.
- The ZAO Fund for Emerging Culture is already dormant ($188 this drive). Adding a second ZAO fund splits
  attention before the first one is even active.

**Revised play:**
1. Submit ZAOstock + Zaoville as projects (kit/make-festivals-on-artizen.md) and curate each into
   We're Loud + Global Music + Greenpill + ZAO Emerging Culture.
2. Put ZOE/Hermes into the Bonfires fund (warm partner).
3. Activate the ZAO fund this drive (rally collects).
4. Only create a ZAO Festivals Fund later IF ZAO wants to be a host/curator for OTHER festival
   organizers (a different, bigger commitment) - and pitch that to René as "the festivals curator,"
   not "the only festivals fund."

The /festivals page stays useful as the ZAO Festivals umbrella's home + an organizer on-ramp; its
positioning is already corrected (run-by-people-who-throw-festivals, not "the only one").
